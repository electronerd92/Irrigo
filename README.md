# Irrigo

This personal IOT project inteds to add some automation to my garden.
The main goal is to set up an automatic system to take care of my future greenhouse and probably a garden bed.
This project uses differents types of Arduino boards, probably some Attiny chip and different sensors.
The power sources will be of different kinds: power supplies and solar panels with LiPo batteries.
The HMI will consist of a LCD menu with a rotary encoder with pushbutton. Moreover some leds and buzzers will be used in order to have alerts.
The system is composed by an Arduino board called Master that will have some others boards acting like actuators or sensors.
The idea is to have the Master connected to a power supply and the other boards powered by the LiPo batteries.
The master will comunicate withe the slaves with cheap Radio Transmitter - Receiver.
